# Python-dev-skills

Python 开发的 AI 编码规范集合，通过科学化的架构设计、错误处理、日志系统、测试策略和代码风格来减少 AI 全自动编写代码犯错的可能。

## 包含内容

- **python-clean-architecture-coder** - Python Clean Architecture 实践指南
  - 层级边界（entities/usecases/adapters/frameworks）
  - 依赖注入与 Repository Protocol
  - 同步/异步架构决策
  - 错误处理与日志规范
  - pytest 测试策略
  - ruff + mypy 代码质量工具

## 适用场景

当你需要编写遵循 Clean Architecture 的 Python 项目时使用此 skill，特别是 FastAPI/API 项目、异步用例、实体建模、适配器模式、依赖注入等场景。
